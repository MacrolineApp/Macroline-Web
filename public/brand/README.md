# Macroline brand assets

Static assets for press, partner integrations, and the Claude Connectors directory submission.

| File | Use |
|---|---|
| `macroline-logo-512.svg` | Primary square logo. 512×512 viewBox with deep navy gradient frame. Use this for connector listings, app store directories, and any context expecting a square logo. |
| `macroline-mark.svg` | Bar-chart mark only, no frame. Use inline in text or in headers where the framed logo is too heavy. |
| `macroline-wordmark.svg` | Horizontal lockup of mark + lowercase "macroline" word. Use on dark backgrounds. |

## Colors used

```
Protein  #F97316  (orange-500)
Carbs    #EAB308  (yellow-500)
Fat      #8B5CF6  (violet-500)

Frame top    #182440
Frame bottom #0A1020
```

## Don't

- Don't recolor the bars; the protein/carbs/fat encoding is part of the brand.
- Don't add a stroke or shadow — the gradient frame already provides depth.
- Don't separate the bars from the baseline rule on the framed logo.
- Don't combine the framed logo with the wordmark text inside the same square — use the wordmark SVG for that lockup.

## Generated by

The framed logo concept is identical to the iOS app icon. The procedural Pillow generator at `Macroline/scripts/generate_icon.py` produces the canonical PNG variants for iOS; these SVGs are the print/web equivalent and were authored by hand to match.
