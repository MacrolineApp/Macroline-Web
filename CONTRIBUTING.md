# Contributing to Macroline-Web

Marketing & support site at macroline.app.

## Workflow

1. Branch from `main`.
2. `npm install`, `npm run dev`, make changes.
3. `npm run build` to verify production build.
4. Open a PR. CI will run a build check.
5. Cloudflare Pages will create a preview deployment automatically.

## What we'll merge

- Copy edits.
- Bug fixes (broken links, layout glitches).
- Accessibility improvements.
- Lighthouse score improvements.

## What we probably won't merge without discussion

- New pages — open an issue first.
- Changes to `/privacy` or `/terms` — these are legal documents.
- Adding heavy client-side JS — keep the site static and fast.

## Voice

Direct, confident about AI, provenance-forward. No hype-y AI marketing. Read existing copy before writing new copy.
